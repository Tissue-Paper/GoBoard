# GoBoard
"Go Board" using JavaScript game library phina.js

##Sample
https://tissue-paper.github.io/GoBoard/example/

## Usage
**Browser**
```html
<script src="https://cdn.rawgit.com/phinajs/phina.js/develop/build/phina.js"></script>
<script src="GoBoard.js"></script>
<script>
  new Board().start();
</script>
```
